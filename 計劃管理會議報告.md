## 三、技術成果重點（管理觀點）

### 量化成果

| 項目 | 數量 |
|:-----|:----:|
| 衛星活動類型涵蓋 | **10 種** |
| 數學限制式 | **66+ 條** |
| 程式核心函式 | **17 個** |
| 測試問題排查並修訂 | **28 項**（第三階段 15、第四階段 13） |
| 微調排程功能 | **5 種** |
| 完整交付文件 | **約 30 份** |

### 質化成果

1. **完整衛星任務排程最佳化系統**：採 IBM ILOG CPLEX 求解器，求解效能經多輪參數調優（8 Threads / Aggressive Cuts / Best-Estimate Search）。
2. **Cold / Hot Scheduling 雙模式架構**：分別處理長期規劃與動態調整需求。
3. **無法排程原因自動偵測機制**：當模型無解時，可自動指出衝突的限制式類別，加速問題排除。
4. **可調式系統參數設計**：透過 `config.properties` 讓操作人員自行調整優先順序、時間窗口、Duty Cycle 上限等。
5. **完整人才移轉**：教育訓練手冊 + 系統操作手冊，確保 TASA 端可獨立運維。

### TASA 113 年度績效認可

本案已列入 **「113 年國家太空中心自主研發的技術」9 項關鍵技術**之一（排程系統最佳化模型開發），為 TASA 對外呈報之年度成果。

> 摘自 113 年技術績效報告：
> > 「進行福衛八號地面排程系統新模組開發，根據衛星元件特性與操作限制，建立模型與限制式……目前已完成第三階段模型優化測試與審查，持續調教模型效能，**未來將與現有系統整合實際應用於福衛八號系列衛星任務**。」
