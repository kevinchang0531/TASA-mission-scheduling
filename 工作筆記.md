## 🔑 技術重點筆記

### BigM 為何設 1,000,000？

MILP 中用「夠大常數 + 二元變數」模擬 If-Then 條件。本案：
- M 需 > 任何時間差最大值
- ACTIVITY Table 保留 5 天 → M ≥ 86400 × 5 = 432,000
- 實務取 1,000,000（約 2.3× 安全係數）
- 第三階段 #7 跨 session 無解 bug 就是 M 設太小造成

### jpa / dao 分層

- `jpa` 套件：定義 `@Entity` 類別，對應資料表（schema 映射）
- `dao` 套件：撰寫 JPQL 查詢，提供業務邏輯呼叫的存取方法
- 文件原文：「**若有新增資料表需求，只要再根據需求創建對應的類別即可**」

### CPLEX 調優參數

- Threads = 8
- Gomory Cuts / Cliques Cuts = Aggressive
- NodeSelect = Best-Estimate Search

## 🕳️ 踩坑筆記

### 編碼問題

- `markitdown` 對中文 PPTX 內容會輸出亂碼（Big5 解讀錯誤）
- 解法：改用 `python-pptx` 直接讀取 + `sys.stdout.reconfigure(encoding='utf-8')`
- `markitdown[docx]` 需另裝依賴；改用 `python-docx` 直接讀

### Preview 面板問題

- `npx serve -s .` 第一次啟動後即使換 launch.json 設定也會被 reuse
- preview 面板路由會吃掉 `.html` 副檔名 → 跳到 `/slides` 而非 `/slides.html`
- 對中文檔名 404 → 複製成 ASCII 檔名 `slides.html` 仍 SPA 模式 404

### GitHub Push API

- `mcp__github__push_files` 對空 repo 會回「Conflict: Git Repository is empty」
- 解法：先用 `git init` + `git push` 建立初始 commit，後續才能用 API
